Sandi’s work

《仁心仁術診療室》作品試圖以臺灣醫療史中的重大怪病——烏腳病為主軸，勾勒1950年代西南部沿海地區歷史，於此架構中編織臺灣人的過往記憶及當代省思。透過數媒網絡時代的語彙提問：「生理上的『烏腳病』雖已不存在，但心理上的『烏腳病』是否真實消失？」

臺灣各民族長期受不同政權殖民的經驗，猶如烏腳病患者反覆經歷截肢、再截肢的命運；現今數位時代線上、線下缺乏隱私的狀態，「非完整的身體」已經成為無可選擇的日常。面對截肢後的身體，身處當代的我們，究竟是有了適合的義肢，得以健全行走，或讓我們以為能跑、能飛的，僅是幻肢？

以滑鼠拖曳的互動翻轉台語「疼痛」、「痛疼」字義，將觀眾帶入場景。 / 螢幕截圖
將敲擊鍵盤的回饋以彈奏風琴的意象呈現，同時帶入隔離期間那些壓抑說不出口的詞。 / 螢幕截圖
